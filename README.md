# KREAM Clone Project

KREAM 서비스를 간단히 모방하여  
웹 개발 전반의 흐름을 학습하고 실제 구현 경험을 쌓기 위한 개인 프로젝트입니다.

프론트엔드, 백엔드, 크롤링, 데이터베이스를 분리된 구조로 설계하여  
웹 서비스의 전체 라이프사이클을 직접 구현하는 것을 목표로 합니다.

본 프로젝트는 기능 완성도보다는  
**아키텍처 설계, 기술 스택 선택, 개발 흐름 이해**에 중점을 두고 진행되었습니다.

## Branch Strategy

- **main**
배포 및 최종 결과물
- **dev**
기능 개발 및 실험을 통합하는 개발 브런치
